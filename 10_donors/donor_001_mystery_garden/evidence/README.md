# Derived Evidence

- This folder is for screenshots, recordings, extracts, and normalized evidence derived from observable donor behavior or raw donor files.
- Every item here should reference the related raw evidence ID or the observed runtime action that produced it.
- Public git keeps evidence indexes, hashes, inventories, and sanitized notes here.
- Raw downloads and runtime payload bodies are excluded from public git and preserved only under `local_only/` on the operator machine.
