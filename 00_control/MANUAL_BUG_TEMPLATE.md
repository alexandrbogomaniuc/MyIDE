# Manual Bug Template

Copy this block into a note or issue:

```md
Date/time:
Local commit SHA:
Feature area:
Local/public/handoff status:

Exact steps:
1.
2.
3.

Expected result:

Actual result:

Screenshot / file paths:

Issue visible in:
- [ ] LOCAL only
- [ ] PUBLIC too
- [ ] HANDOFF / publish flow

Extra notes:
```

## Evidence Checklist
- One screenshot or exact file path
- The local commit SHA from `npm run manual:status`
- The exact step number where the issue appeared
- Whether the issue is only local or also present in public state
